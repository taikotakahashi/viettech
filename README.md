# White Devs, Inc. — Landing Page

A pixel-faithful React + Vite landing page for **White Devs, Inc.**, matching the provided UI designs with full responsive support.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite 8**
- **Tailwind CSS 4**
- **Lucide React** (line icons)
- **React Icons** (brand/tech logos)

## Getting Started

```bash
cd white-devs-site
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```

## Responsive Breakpoints

| Breakpoint | Layout behavior |
|---|---|
| **320px – 374px** | Single-column stacks, vertical hero stats, tight 16px padding |
| **375px – 424px** | 2×2 hero/results grids, improved type scale |
| **425px – 767px** | Side-by-side CTAs, 2-column project cards, About two-column |
| **768px – 1023px** | 4-column stats, 3-column process grid |
| **1024px+** | Full desktop nav, 3-column projects, 6-step process timeline |
